# Roadmap

## Corto plazo

- consolidar la estructura nueva del repositorio,
- migrar scripts y tooling a modulos estables,
- normalizar outputs de discovery y catalogo,
- versionar mejor las tablas base de Supabase.

## Siguiente hito tecnico

- encapsular integraciones Mercantil en cliente dedicado,
- migrar la logica de cotizador a `backend/services/`,
- definir contratos internos para cotizacion y catalogo,
- mantener compatibilidad con el frontend actual.

## Siguiente hito de negocio

- conectar cotizacion real en lugar de precio fijo,
- habilitar seguimiento comercial automatico,
- medir conversion por paso y origen de trafico,
- priorizar leads con mayor potencial.

## Hitos posteriores

- incorporar tablas maestras de usos y localidades,
- agregar mensajeria operativa por WhatsApp,
- sumar agente de prevalidacion con IA,
- completar migracion desde `aplicacion/` hacia la nueva arquitectura productiva.
